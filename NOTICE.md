## üìñ What This Is

This repository showcases the **backend architecture** of ASTRAL AI, a production AI system that:

- Builds intelligent Minecraft modpacks using Large Language Models
- Performs semantic search across 100,000+ mods
- Autonomously analyzes and fixes crash logs
- Serves real users with authentication and rate limiting

**Key Technologies:**
- Python 3.11+, Flask 3.0
- DeepSeek LLM (GPT-compatible API)
- PostgreSQL + pgvector (vector database)
- Supabase (auth + database)
- sentence-transformers (embeddings)
